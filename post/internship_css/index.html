CSS

Web pages are buillt and structured with HTML, which specifies the content and hirachy of a page. CSS(Cascading Styles Sheets) specifies the apperance of a page.

CSS is a set of static rules that define how an object appears on the page based on its selectors. Styles / Rules are only applied to elements if they have the correct selector and thus follow under the given rule.

Comments

    /*
        This is a comment
    */

Selectors

Different types of selectors are given a different prefix in order to be understood in the context of css

    /* A class */
    .class
    /* An id */
    #id
    /* Its name is also a selector */
    div
    /* 
        One may also chain multiple selectors 
        The code below would effect both classes
    */
    .class1.class2 { }

    /* 
        Elements can also be selected based on their attributes
    */
    [attr] { font-size:smaller; }

    /* 
        Or that they have a specific attribute value
    */
    [attr='value'] { font-size:smaller; }

    /* 
        You can then also chain these different selectors to get a very specific class
    */
    div.some-class[attr$='ue'] { }

    /* 
        CSS also has a model of "inheritance"
        Therefore we can also select parents and children
    */
    div.some-parent > .class-name { }

    div.some-parent .class-name { }

Events

CSS has a set of special selectors that can be used to select an element based on a particular state

    /* When an element is hovered over */
    selector:hover { }

    /* When an element has been visited {like a link} */
    selector:visited { }

    /* When an element has not been visited */
    selected:link { }

    /* An element is focused */
    selected:focus { }

You can also use some locational selectors to select elements that have a certain place in the document

    /* The first child */
    selector:first-child {}

    /* The last child */
    selector:last-child {}

    /* Every nth element */
    selector:nth-child(2)

    /* Every odd element */
    p:nth-child(odd) {
        background: red;
    }

    /* Every even element */
    p:nth-child(even) {
        background: blue;
    }

Some special selectors include

    /* All elements */
    * { }

    /* All descendants */
    .parent * { }

    /* All children */
    .parent > * { }

Some ways of setting sizes in CSS

    width: 50%;/* percentage of parent element width */
    font-size: 2em;   /* multiples of element's original font-size */
    font-size: 2rem;  /* or the root element's font-size */
    font-size: 2vw;   /* multiples of 1% of the viewport's width (CSS 3) */
    font-size: 2vh;   /* or its height */
    font-size: 2vmin; /* whichever of a vh or a vw is smaller */
    font-size: 2vmax; /* or greater */
