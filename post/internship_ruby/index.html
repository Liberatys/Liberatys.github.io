Today we had the opportunity to go through a tutorial / course on the ruby programming language. Because Ruby on Rails is a wiedly adopted tool in the field of web development, ruby and the knowledge thereof can come in handy in many situations.

Difference between Ruby and other langauges

  • Ruby is a very dynamic and adaptive programming language that enables the programmer to go fast and stay close to the meaning of what he is trying to achieve
  • Most methods and operations in ruby are intutive and very close to spoken language(English)
  • Ruby code is compact, elegant and brings along a lot of hidden magic
  • Ruby is an open source langauge. Therefore anyone working with the langauge can report issues and fix them right away. This results in a very user orientated langauge that is a compilement of many great ideas sourcing from many great minds

Variables and Datatypes

Like most programming languages ruby has different datatypes that can be stored in variables. The most used and basic ones are:

  • Numbers
  • Booleans
  • Strings

In ruby a type decleration is not needed, therefore we can just asign values to variables without knowing what type it is (kinda)

    my_num = 25
    my_boolean = true
    my_string = "Ruby"
        
    puts my_num
    puts my_boolean
    puts my_string

“puts” prints the value of the variable to the standard output
