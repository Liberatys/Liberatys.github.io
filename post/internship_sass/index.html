SASS

Sass is an extension to the css language. It tries to bring some features to css that will make larger codebases easier and will also help developers keep track of current colors and other variables that are the same all across the board.

It is a tool to write css faster and make its mantenance easier and cleaner.

Variables

$main-color: #27d79d;
$font: 'Roboto', sans-serif;


// It is then possible to reference these variables in other parts of the code
body {
    background-color: $main-color;
    font-family:  $font;
}

Contro Flow

Like other “programming languages” SCSS lets you use if, else, while and each to control the flow of your code compilation

$goodday: true !default;
@mixin goodday {
    @if $goodday {
        display: inline-block;
    }
    @else {
        display: none;
    }
}
.info {
    @include goodday;
}



// the code below willl create div:nth-of-type(1-100)
@for $c from 1 to 100 {
    div:nth-of-type(#{$c}) {
        left: $c;
    }
}
==>

div:nth-of-type(1) {
   left: 1;
}
 div:nth-of-type(2) {
   left: 2;
}

// @each is used in the same way as for but requires a list the be used on

$list: 1 2 3 4 5;
.numbers {
    @each $sm in $list {
        .number-#{$sm} {
            font-size: $sm;
        }
    }
}

// mixins are placeholders that stand for a set of set stylings that can be imported into other stylings

@mixin center {
    display: block;
    margin-left: auto;
    margin-right: auto;
    left: 0;
    right: 0;
}

div {
    @include center;
    background-color: $primary-color;
}


// mixins can also be used with variables
@mixin big($width, $height) {
    width: $width;
    height: $height;
}

.square {
    @include size(60px, 60px);
}
