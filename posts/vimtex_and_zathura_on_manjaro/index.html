18.04.2021
tools , linux , nvim

Installing Vimtext with Zathura on Manjaro / Arch Linux

A short introduction to installing vimtex and zathura on linux (arch / manjaro).

Installations

First, you should install zathura on your system

I usually install packages from the integrated package manager but you are free to install them from the source.

    sudo pacman -S zathura

To use vimtex we also need latexmk which is usually packaged inside the bigger latex utilities and applications. An easy way to get it is to install texlive-core from pacman.

    sudo pacman -S texlive-core

Setup in Neovim

Create a file in your $HOME/.config/nvim folder

    touch $HOME/.config/nvim/latex.vim

and then source this file in your nvim.init file

    source $HOME/.config/nvim/latex.vim

You don’t have to put this into a separate file! But I find it easier to maintain my config that way.

Installing Vimtex itself

Head to https://github.com/lervag/vimtex and follow the installation instruction. And please leave some love in the form of a star :D.

This is also a great opportunity to install a package manager for nvim if you haven’t already done so! (Have a look at https://github.com/junegunn/vim-plug)

Installing some nvim packages with python

The following packages are also needed for nvim to enable:

  1. Controlling nvim from the shell

  2. Opening files from within :terminal without nesting!

     pip3 install --user pynvim neovim-remote

Configure Vimtex

We first need to setup some stuff so that vimtex knows about the communication and type of tex to use:

    let g:tex_flavour = 'latex'
    let g:vimtex_compiler_progname = 'nvr'
    let g:vimtex_view_automatic = 1

If you intend to use vimtex and neovim on different systems it may be of interest to you to use a logic check to set up for different systems.

    if has('unix')
        if has('mac')

            else

        endif

        elseif has('win32')
            # WHY? WHY? STOP IT!
    endif

Now set up your zathura with vimtext

    let g:latex_view_general_viewer = 'zathura'
    let g:vimtex_view_method = 'zathura'

Using Vimtex

Head to a .tex document of your choice and use :VimtexCompile in nvim and your .tex document should be compiled and also displayed within Zathura!

A very short intro into installing vimtex and zathura on linux!
